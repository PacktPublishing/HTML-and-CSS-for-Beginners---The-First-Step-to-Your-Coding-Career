Selector What does it do
p select all elements named: p
.foo select all elements that class='foo'
#id-name select all elements that id='id-name'
div, p select all elements that are div and p
div p select all p INSIDE OF div.
div .foo select all class='foo' INSIDE OF div.
div p.foo select all p, which class='foo' - INSIDE OF div.
