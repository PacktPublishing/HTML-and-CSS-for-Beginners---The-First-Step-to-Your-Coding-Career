The Basics of Page Creation - Brief Introduction

What is HTML?

HTML stands for Hypertext MarkUp Language. It is a tag language.

What are tags?

Tag is a marking, it is used to select text. It gives the text a predetermined meaning.

What are attributes?

Attribute is additional information about the tag. Some tags may need to include an additional tag attribute. Thanks to the attributes, we are able to add such a feature.

What is CSS?

What is a selector?

A selector is an operator that allows you to select (choose) specific tags from a page that interest you in order to style them later (make them look different).

How do I include a CSS file in a page?

To attach a file to a page, use the <link> tag in the head section.
